# Adventure Investment Game

A console-based adventure game developed using Python. The player explores different locations, plays mini-games, earns gold, and manages health to complete the adventure.

## Features
- Multiple chapters (Forest, Village, Castle)
- Mini-games:
  - Dice Betting Game
  - Number Guessing Game
  - Rock Paper Scissors
- Inventory and health management
- Randomized outcomes for replayability

## Technologies Used
- Python
- Random module

## How to Run
1. Install Python (version 3.x)
2. Open terminal or command prompt
3. Run the file using:
   ```bash
   python adventure_game.py
